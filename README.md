# MariaDB Foundation — `.github`

This repository hosts community health files for the MariaDB Foundation GitHub organization, along with official brand assets for external use.

## Logo & Branding

The MariaDB Foundation logo is provided in [`assets/logos`](./assets/logos), available in the following formats:

- **SVG** (`assets/logos/svg`) — scalable, recommended for web use
- **PNG** (`assets/logos/png`) — for quick embeds and use cases where SVG isn't supported

These formats cover the vast majority of use cases, including web embeds, print, and social previews.

### Usage Guidelines

- These marks identify the **MariaDB Foundation**. You may use them to refer to the Foundation and its activities — for example, to link to us, to indicate membership or sponsorship, or in articles, talks, and presentations about the Foundation.
- Use the files as provided. Don't alter the proportions, colours, or elements. Where you need a single-colour or reversed version for dark backgrounds, use the variants we supply rather than recolouring it yourself.
- Don't use the logo to suggest that the Foundation endorses, sponsors, or is affiliated with your product, project, or organisation unless we have agreed to that.
- Don't use it to refer to other MariaDB or non-MariaDB entities, and don't incorporate it into your own logo, product name, or branding.
- Give it clear space; don't lock it up with other marks so they read as one combined mark.

### Source Files

Original Adobe Illustrator (`.ai`) source files are not included in this repository.

If you require the source files — for example, for custom resizing, color adjustments, or print production — please contact the MariaDB Foundation at **foundation@mariadb.org** with details on your intended use.
